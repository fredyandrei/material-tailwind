### Installation

Learn how to use Horizon UI components from this documentation to quickly and easily create elegant and flexible pages using Tailwind CSS.

<br />

### NPM

Make sure that you have [Node.js](https://nodejs.org/en/) and [Tailwind CSS](https://tailwindcss.com/) installed.

<br />
Install Horizon UI as a dependency using NPM by running the following command:

```html showLineNumbers
npm i @horizon-ui-main/html
```

<br />
Require Horizon UI as a plugin inside the **tailwind.config.js** file:

```html showLineNumbers
module.exports = { plugins: [ require('horizon-ui-main/plugin') ] }
```

<br />
Include the main JavaScript file to make interactive elements working:

```html showLineNumbers
<script src="../path/to/horizon-ui-main.js"></script>
```

<br />
### Yarn{" "}

```html showLineNumbers
yarn add @horizon-ui-main/html
```

<br />

### PNPM

```html showLineNumbers
pnpm i @horizon-ui-main/html
```

<br />

### Import Horizon UI Style Sheet

Import the **tailwind.css** into your project file. Make sure to import the tailwind.css style sheet after all other style sheets.

```html showLineNumbers
<link rel="stylesheet" href="styles.css" />
```

<br />

### Link Material Icons Fonts to your project

Add a link for the material icons fonts inside the head of your project, so to use the material icons with Horizon UI components. You can also use font awesome font family for the project just add a link for it inside the head of your project.

```html showLineNumbers
// Material Icons Link
<link
  href="https://fonts.googleapis.com/icon?family=Material+Icons"
  rel="stylesheet"
/>

// Font Awesome Link
<link
  rel="stylesheet"
  href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.2/css/all.min.css"
  integrity="sha512-HK5fgLBL+xu6dm/Ii3z4xhlSUyZgTT9tuc/hSrtw6uzJOvgRr2a9jyxxT1ely+B+xFAmJKVSTbpM/CuL7qxO8w=="
  crossorigin="anonymous"
/>
```
