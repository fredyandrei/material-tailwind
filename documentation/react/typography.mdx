# Tailwind CSS Typography - React

Use our Tailwind CSS typography to present your website/web app as clearly and efficiently as possible.

Below we are presenting a simple typography component example. It comes in different styles and colors, so you can adapt it easily to your needs.

<br />

<ComponentCard id='typography'>
  <Typography>
    {`Horizon UI is an easy to use components library for Tailwind CSS and
    Material Design. It provides a simple way to customize your components, you
    can change the colors, fonts, breakpoints and everything you need.`}
  </Typography>
</ComponentDemo>

```jsx showLineNumbers
import { Typography } from "@material-tailwind/react";

export default function Example() {
  return (
    <Typography>
      Horizon UI is an easy to use components library for Tailwind CSS and
      Material Design. It provides a simple way to customize your components,
      you can change the colors, fonts, breakpoints and everything you need.
    </Typography>
  );
}
```

## Props

<span id='props' className='scroll-mt-48' />
The following props are available for the typography component. These are the custom
props that come with we've added for the typography component and you can use all
the other native props as well.

| Attribute      | Type                       | Description                                   | Default                                |
| -------------- | -------------------------- | --------------------------------------------- | -------------------------------------- |
| `variant`      | [Variant](#types)          | Change typography variant                     | <Code small>paragraph</Code>           |
| `color`        | [Color](#types)            | Change typography color                       | <Code small>inherit</Code>             |
| `textGradient` | <Code small>boolean</Code> | Change typography color into a gradient color | <Code small>false</Code>               |
| `as`           | <Code small>element</Code> | Change the typography rendered element        | <Code small>undefined</Code>           |
| `className`    | <Code small>string</Code>  | Add custom className for typography           | `''`                                   |
| `children`     | <Code small>node</Code>    | Add content for typography                    | No default value it's a required prop. |
