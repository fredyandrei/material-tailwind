# Tailwind CSS Badge - React

Get started on your web projects with our Tailwind CSS badge which is a compact elements that represent an input, attribute, or action. This element appears dynamically as a group of multiple interactive elements and allows users to enter information, make selections, filter content, or trigger actions.

See below our simple badge component example that you can use for your Tailwind CSS and React project. The example also comes in different styles and colors, so you can adapt it easily to your needs.

<br />

<ComponentDemo id='badge'>
  <Badge value='badge' />
</ComponentDemo>

```jsx showLineNumbers {1, 4}
import { Badge } from "@material-tailwind/react";

export default function Example() {
  return <Badge value='badge' />;
}
```

## Badge Variants

The badge component comes with 2 different variants that you can change it using the <Code>variant</Code> prop.

<ComponentDemo id='badge-variants'>

    <Badge variant='gradient' value='badge gradient' />
  </div>
</ComponentDemo>

```jsx showLineNumbers {6-7}
import { Badge } from "@material-tailwind/react";

export default function Variants() {
  return (
    <div className='flex gap-2'>
      <Badge variant='filled' value='badge filled' />
      <Badge variant='gradient' value='badge gradient' />
    </div>
  );
}
```

## Badge Dismissible

The badge component is a dismissible component that you can control it using the <Code>dismissible</Code> and <Code>show</Code> props.

<State initialState={{ show: true }}>
  <ComponentDemo id='badge-dismissible'>
    <Observe>
      {({ setState, show }) => (
        <>
          {!show && (
            <Button
              variant='gradient'
              className='absolute'
              onClick={() => setState((s) => ({ show: true }))}>
              Show Badge
            </Button>
          )}
          <Badge
            variant='gradient'
            show={show}
            dismissible={{
              onClose: () => setState((s) => ({ show: false })),
            }}
            value='Dismissible'
          />
        </>
      )}
    </Observe>
  </ComponentDemo>
</State>

```jsx showLineNumbers {5, 8-26}
import { useState, Fragment } from "react";
import { Badge } from "@material-tailwind/react";

export default function Dismissible() {
  const [show, setShow] = useState(true);

  return (
    <Fragment>
      {!show && (
        <Button
          variant='gradient'
          className='absolute'
          onClick={() => setShow(true)}>
          Show Badge
        </Button>
      )}
      <Badge
        variant='gradient'
        show={show}
        dismissible={{
          onClose: () => setShow(false),
        }}
        value='Dismissible'
      />
    </Fragment>
  );
}
```

## Props

<span id='props' className='scroll-mt-48' />
The following props are available for the button component. These are the custom
props that come with we've added for the badge component and you can use all the
other native props as well.

| Attribute     | Type                       | Description                        | Default                               |
| ------------- | -------------------------- | ---------------------------------- | ------------------------------------- |
| `variant`     | [Variant](#types)          | Change badge variant               | <Code small>filled</Code>             |
| `color`       | [Color](#types)            | Change badge color                 | <Code small>blue</Code>               |
| `value`       | [Value](#types)            | Add content for badge              | No default value its a required prop. |
| `dismissible` | [Dismissible](#types)      | Makes the badge dismissible        | <Code small>undefined</Code>          |
| `animate`     | [Animate](#types)          | Change badge animation             | <Code small>undefined</Code>          |
| `show`        | <Code small>boolean</Code> | Change the badge visibility        | <Code small>true</Code>               |
| `icon`        | <Code small>node</Code>    | Add icon at the beginning of badge | <Code small>undefined</Code>          |
| `className`   | <Code small>string</Code>  | Add custom className for badge     | `''`                                  |
